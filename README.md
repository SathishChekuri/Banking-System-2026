# Banking Management System

A full-stack Banking Management System using Spring Boot, MySQL, Thymeleaf, HTML, CSS and Java.

## Features

- Create Account
- Deposit
- Withdraw
- Transfer
- Balance Check
- Transaction History
- KYC Update
- Admin Dashboard
- Block/Unblock Accounts

## Technologies

- Java
- Spring Boot
- MySQL
- Thymeleaf
- HTML/CSS
