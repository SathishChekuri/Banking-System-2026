# Banking Management System

A full-stack Banking Management System built using Spring Boot, Java, MySQL, Thymeleaf, HTML, and CSS.  
This project simulates real-world banking operations with secure account management, money transactions, KYC updates, and admin controls.

## Features

### User Module
- Create Account
- Auto-generated Account Number
- Auto-generated Password
- Deposit Money
- Withdraw Money
- Transfer Money
- Balance Check
- Transaction History
- KYC Update

### Admin Module
- Secure Admin Login
- Banking Statistics Dashboard
- View All Users
- View User Transactions
- KYC Status Management
- Block / Unblock Accounts

## Technologies Used

- Java
- Spring Boot
- MySQL
- Thymeleaf
- HTML
- CSS
- Bootstrap
- Maven
