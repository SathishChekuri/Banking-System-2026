# Banking Management System

A full-stack Banking Management System developed using Spring Boot, Java, MySQL, Thymeleaf, HTML, CSS, and Bootstrap.

This project simulates real banking operations with secure account management, transactions, KYC updates, and admin controls.

---

## Features

### User Features
- Create Bank Account
- Auto Account Number Generation
- Auto Password Generation
- Deposit Money
- Withdraw Money
- Transfer Money Between Accounts
- Balance Check
- View Transaction History
- Update KYC Details

### Admin Features
- Secure Admin Login
- Banking Statistics Dashboard
- View All Users
- View User Transactions
- Approve/Update KYC Status
- Block / Unblock Accounts

## How To Run

1. Clone the repository

```bash
git clone YOUR_GITHUB_REPOSITORY_LINK
```

2. Open project in VS Code / IntelliJ

3. Configure MySQL database in:

```text
application.properties
```

4. Run the project

```bash
mvn spring-boot:run
```

5. Open browser

```text
http://localhost:8080
```
